# Forecasting India's Quarterly Gross Value Added (GVA) Using ARIMA Models

## Project Overview
This project analyzes and forecasts India's quarterly GVA data from 2011 to 2023 using an ARIMA(1,0,0) model, with a specific focus on post-pandemic economic dynamics and policy implications.

## Key Findings
*   The ARIMA(1,0,0) model was selected as the best fit.
*   Forecasts predict a steady increase in GVA throughout 2024.
*   A strong positive autoregressive coefficient indicates that past performance significantly influences future output.

## Methodology
*   **Data Source:** Ministry of Statistics and Programme Implementation (MOSPI)
*   **Software:** R (using `forecast`, `tseries`, and `readxl` packages)
